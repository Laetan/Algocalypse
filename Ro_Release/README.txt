Compilable situé dans le répertoire build
Programme ne fonctionnant que sous WINDOWS

Pour personaliser le graphe, il faut modifier le fichier Ressources\param.txt

Le premier groupe de chiffre représente la matrice d'adjacence.

Le deuxième indique les contraintes supérieures des arcs.

Les éléments d'une matrice de la même ligne doivent être séparés par un point-virgule (;). La ligne ne doit pas finir par un point-virgule
